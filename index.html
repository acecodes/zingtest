Who's on top?

Boring old Java, that's who.
The language that just won't die.

Java, the workhorse of enterprise applications the world over, is still the reigning champion of the programming language popularity contest. The rising stars of more streamlined and modern languages such as Python and JavaScript still cannot contend with the sheer quantity of Java code that is floating out in cyberspace. Every time the term "J2EE" is used, a Node engineer dies a little inside.

Data provided by TIOBE in August 2015.


...Does that mean I should learn Java?

You can make a comfortable living with almost any major language.

Focus on what you enjoy and are good at.

If Java and C programmers can still get paid this well, you can bet that any of the major languages are going to continue being popular for the foreseeable future. So take a deep breath and remember: if you can program, you're still valuable. Period. And let's face it, programming is a much better way to earn a paycheck than digging ditches. Trust me, I know from first-hand experience.
